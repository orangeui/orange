# Font

Orange's default typeface is <mark>Rubik</mark>, designed by Philipp Hubert and Sebastian Fischer at Hubert & Fischer.

```scss
.example {
  font-family: $o-font-primary;
}
```

<hr/>


# Tryout

Tryout is for live example of Font on **Canvas** tab and bottom tab **Knobs**

<Story name='Tryout'>
  {{
    components: { },
    template: `<p :class="classes">
      {{ fontWeight }}
    </p>`,
    props: {
      fontWeight: {
        default: select('Font weight', ['normal', 'medium', 'bold', 'extrabold'], 'normal')
      },
      textItalic: {
        default: boolean('Text italic', false)
      }
    },
    computed: {
      classes () {
        return {
          [`font-${this.fontWeight}`]: this.fontWeight,
          [`text-italic`]: this.textItalic
        }
      }
    }
  }}
</Story>

# Weights

Apply weights by adding <mark>.font-{weight}</mark> class to text elements

Recall weights with the <mark>fontWeight('{weight}')</mark> function

<Story name='Weights'>
  {{

    template: `<ul class="docs__simple-list">

      </li>
      <li>
        <h5>600 Bold</h5>
        <p class="fs-gamma font-bold">600 Bold</p>
      </li>
      <li>
        <h5>500 Medium</h5>
        <p class="fs-gamma font-medium">500 Medium</p>
      </li>
      <li>

      </li>
    </ul>`,

```html
<p class="font-extrabold">700 Extra Bold</p>
<p class="font-bold">600 Bold</p>
<p class="font-medium">500 Medium</p>
<p class="font-normal">400 Normal</p>
```

```scss
.example {
  font-weight: fontWeight('extrabold');
}
```
