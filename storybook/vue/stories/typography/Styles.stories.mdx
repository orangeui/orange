# Styles

Apply a color by adding <mark>.color-{color}</mark> class

Recall color values with the <mark>color('{color}')</mark> function

<hr />

# Tryout

Tryout is for live example of Styles on **Canvas** tab and bottom tab **Knobs**

<Story name='Tryout'>
  {{
    components: { },
    template: `<p :class="classes">
      Suspendisse id nisi a magna lacinia lacinia sed sed purus.
    </p>`,
    props: {
      color: {
        default: select('Color', ColorSlugs, ColorSlugs[1])
      },
      heading: {
        default: select('Heading', [null, 'h1', 'h2', 'h3', 'h4', 'h5', 'h6'], null)
      }
    },
    computed: {
      classes () {
        return {
          [`color-${this.color}`]: this.color,
          [`${this.heading}`]: this.heading,
        }
      }
    }
  }}
</Story>
