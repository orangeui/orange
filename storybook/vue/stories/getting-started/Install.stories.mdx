# Orange CSS

Add stylesheets and scripts to all your pages via our CDN.

```html
<link rel="stylesheet" href="https://cdn.orangecss.com/1.4.2/css/orange.min.css">

```

```html
<script src="https://cdn.orangecss.com/1.4.2/js/orange.min.js"></script>
```

The latest release can be found at https://github.com/orangeui/orange.
