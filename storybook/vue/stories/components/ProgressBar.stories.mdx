# Progress Bar

A progress bar informs users about the progression of a running task, and can contextually mark the current point in a period.

The Progress Bar component is a custom component and comes in line and circular style, both of which offer different levels of data exposure.

<hr/>


# Tryout

Tryout is for live example of progress bar on **Canvas** tab and bottom tab **Knobs**

<Story name='Tryout'>
  {{
    components: { OProgressBar },
    template: `<o-progress-bar class="mb-4" :percentage="percentage" :type="type" :color="color" :text_inside="text_inside" :show_text="show_text" :custom_text="custom_text" :stroke_width="stroke_width" :disabled="disabled"></o-progress-bar>`,
    props: {
      type: {
        default: select('Type', ['line', 'circle'], 'line')
      },
      percentage: {
        default: number('Percentage', '15')
      },
      text_inside: {
        default: boolean('Text inside', false)
      },
      show_text: {
        default: boolean('Show text', false)
      },
      custom_text: {
        default: text('Custom text', 'some text')
      },
      color: {
        default: text('Color', 'green')
      },
      stroke_width: {
        default: number('Stroke width', 8)
      },
      disabled: {
        default: boolean('Disabled', false)
      }
    },
  }}
</Story>
