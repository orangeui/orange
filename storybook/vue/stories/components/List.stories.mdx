# List

A list is <mark>a container</mark> that <mark>groups items</mark>.

When wrapping a list with a <mark>.card</mark> element use the <mark>no_gutter</mark> attribute that overflows card's body gutter. This makes the list take full width of it's parent card.

You can add horizontal dividers between list items by specifying <mark>divide</mark> attribute.

Add a hover effect to a list with <mark>hover</mark> attribute.
