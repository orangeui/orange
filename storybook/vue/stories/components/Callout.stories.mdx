# Tryout

Tryout is for live example of Callout on **Canvas** tab and bottom tab **Knobs**

<Story name='Tryout'>
  {{
    components: { OCallout },
    template: `<o-callout :color="color" :shadow="shadow" :action="action" @click="handleClick">
      Orange color lies between red and yellow on the light spectrum
    </o-callout>`,
    methods: { handleClick: action('click') },
    props: {
      color: {
        default: select('Color', ColorBase, null)
      },
      shadow: {
        default: boolean('Shadow', false)
      },
      action: {
        default: boolean('Action', false)
      },
    },
  }}
</Story>
