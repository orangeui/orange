# Display

Display class define display class of elements.

<hr />

Apply display to elements with <mark>.d-{key}</mark> class

Recall display with <mark>display({key})</mark> function in <mark>SCSS</mark>

```scss
$displays: (
  'block': block,
  'inline-block': inline-block,
  'inline': inline,
  'flex': flex,
  'inline-flex': inline-flex,
  'table': table,
  'table-cell': table-cell,
  'grid': grid,
  'none': none
) !default;
```
