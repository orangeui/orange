# Spacings

Values that define empty space between elements.

<hr />

Apply margin to elements with <mark>.m-{key}</mark>, <mark>.ml-{key}</mark>, <mark>.mr-{key}</mark>, <mark>.mt-{key}</mark>, <mark>.mb-{key}</mark> classes

Apply padding to elements with <mark>.p-{key}</mark>, <mark>.pl-{key}</mark>, <mark>.pr-{key}</mark>, <mark>.pt-{key}</mark>, <mark>.pb-{key}</mark> classes

Recall spacings with <mark>space({key})</mark> function in <mark>SCSS</mark>

```scss
$o-space-base: 24px;

$o-spaces: (
  0: 0,                 // 0px
  1: $space-base / 6,   // 4px
  2: $space-base / 3,   // 8px
  3: $space-base / 2,   // 12px
  4: $space-base,       // 24px
  5: $space-base * 2,   // 48px
  6: $space-base * 3,   // 72px
  7: $space-base * 6    // 144px
);
```

# Responsive

Apply responsive margin to elements with <mark>.{breakpoint}:m-{key}</mark>, <mark>.{breakpoint}:ml-{key}</mark>, <mark>.{breakpoint}:mr-{key}</mark>, <mark>.{breakpoint}:mt-{key}</mark>, <mark>.{breakpoint}:mb-{key}</mark> classes
