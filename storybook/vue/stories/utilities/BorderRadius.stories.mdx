# Border Radius

Values that define border radius of element.

<hr/>

Apply border radius to elements with <mark>.rounded-{key}</mark> class

Recall border radius with <mark>rounded({key})</mark> function in <mark>SCSS</mark>

```scss
$o-border-radius-base: 3px;

$o-border-radiuses: (
  0: 0,                         // 0px
  'base': $o-border-radius-base,  // 3px
  1: $o-border-radius-base * 2,   // 6px
  'full': 50%                   // 50%
);
```

# Tryout

Tryout is for live example of Border Radius on **Canvas** tab and bottom tab **Knobs**

<Story name='Tryout'>
  {{
    components: { },
    template: `<div>
      <img class="image" :class="classes" src="https://orangecss.com/images/thumbnails/square/fox-square.jpg" alt="Foxy" />
    </div>`,
    props: {
      rounded: {
        default: select('Rounded', ['rounded-0', 'rounded', 'rounded-1', 'rounded-full'], 'rounded')
      }
    },
    computed: {
      classes () {
        return {
          [`${this.rounded}`]: this.rounded,
        }
      }
    }
  }}
</Story>
